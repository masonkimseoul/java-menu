# 구현 기능 목록
### 메뉴 추천 기능
1. 카테고리 정하기
   1. 랜덤으로 선정
      1. 1 - 일식, 2 - 한식, 3 - 중식, 4 - 아시안, 5 - 양식
   2. 같은 카테고리는 한 주에 최대 2회 선정 가능
2. 1.에서 정한 카테고리에서 먹을 메뉴 정하기
   1. 랜덤으로 선정
      1. 각 카테고리의 메뉴를 섞고, 가장 처음 메뉴를 추천
   2. 못 먹는 메뉴 제외
      1. 0 ~ 2개 메뉴
   3. 한 사람의 추천 리스트에 중복된 메뉴는 없어야 함
3. 월 ~ 금 반복

### 입출력 기능
1. 입력
   1. 코치의 이름 입력
      1. 2 ~ 4글자
      2. 2 ~ 5명 (','로 분리)
   2. 각 코치마다 못 먹는 메뉴 입력
      1. 0 ~ 2개 (','로 분리)
         1. 다 잘 먹으면 빈 값
   
2. 출력
   1. 안내 문구 출력
   2. 코치들의 이름 출력
   3. 각 코치마다 못 먹는 메뉴 출력
   4. 메뉴 추천 결과 출력

### 예외 처리 기능
1. 코치 이름 입력 시 예외
   1. 코치의 이름이 2글자 미만, 4글자 초과 시 예외
   2. 입력한 코치의 수가 2명 미만, 5명 초과 시 예외
2. 못 먹는 메뉴 입력 시 예외
   1. 코치 당 못 먹는 메뉴가 2개 초과 시 예외
   2. 못 먹는 메뉴가 존재하지 않는 메뉴일 때 예외
